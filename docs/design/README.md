# Collectium designregler

Denne mappen er root for Collectium sine globale designregler, designkilder, template-regler, AI-indeks og visuelle styringsdokumenter.

AI/Codex skal lese denne mappen før den lager eller endrer:
- layout
- template
- skins
- farger
- typografi
- rammer
- signatur
- globale komponenter
- sideoppsett

Vanlige sider skal ikke eie design. Design skal styres globalt fra template/designmotor og CSS tokens.
